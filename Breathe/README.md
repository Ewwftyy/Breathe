## Breathing Technique

The app guides you through the 4-4-4-4 breathing pattern:

1. **Inhale (Nose)** - 4 seconds
2. **Hold** - 4 seconds  
3. **Exhale (Mouth)** - 4 seconds
4. **Hold** - 4 seconds

This cycle repeats continuously to help you maintain a calm, steady breathing rhythm.

```bash
git clone https://github.com/Ewwftyy/Breathe.git

```

2. Run the application:
```bash

```

## Usage

1. Launch the application
2. Click the **Start** button to begin the breathing exercise
3. Follow the on-screen instructions and countdown
4. Click **Stop** to pause at any time
5. Click **Reset** to start over from the beginning
